# Stolpejakt-slack

Integration between Stolpejakt and Slack

## Environment file

Necessary environment variables for your `.env` file

```dotenv
TOKEN="<Bearer token from Stolpejaktens website>"
WEBHOOK_URL="<Webhook url to a slack bot>"
```
